# adminPro-NodeJs
adminPr BackEnd with NodeJs

Este es el codigo necesario para establecer el backend conectado a MongoDB usando Mongoose.

Para ejecuarlo, es necesario reconstruir los modulos de node usando el comando

```
npm install
```
